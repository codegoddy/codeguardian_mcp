DevHQ CLI Installation

Time tracking from your terminal

🍎 macOS

curl -fsSL https://www.devhq.site/cli/install-macos.sh | sh

🐧 Linux

curl -fsSL https://www.devhq.site/cli/install.sh | sh

🪟 Windows (PowerShell)

iwr -useb https://www.devhq.site/cli/install-windows.ps1 | iex
After installation:
1. Get your API token from devhq.site/integrations
2. Configure: devhq config set api-token YOUR_TOKEN
3. Start tracking: devhq start TRACKING_CODE

For more information, visit www.devhq.site/download
